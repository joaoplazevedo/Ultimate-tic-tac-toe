# Ultimate Tic Tac Toe (Java) - Jogo do Galo

Implementação simples do **Ultimate Jogo do Galo** em Java, com duas formas de jogar:
- Terminal
- GUI (Feito com Swing)

## Como Jogar
Escrever no terminal:
  - git clone <ainda nao pensei o nome do repo>
  - javac *.java && java Launcher
